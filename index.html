task1.Дана строка 'ahb acb aeb aeeb adcb axeb'. Напишите регулярку, которая найдет строки ahb, acb, aeb по шаблону: буква 'a', любой символ, буква 'b'.
task2.Дана строка 'aba aca aea abba adca abea'. Регулярка, которая найдет строки abba adca abea по шаблону: буква 'a', 2 любых символа, буква 'a'.
task3 Дана строка 'aba aca aea abba adca abea'. Регулярка, которая найдет строки abba и abea, не захватив adca.
task4. Дана строка 'aa aba abba abbba abca abea'. Регулярка, которая найдет строки aba, abba, abbba по шаблону: буква 'a', буква 'b' любое количество раз, буква 'a'
task5.Дана строка 'aa aba abba abbba abca abea'. Регулярка, которая найдет строки aa, aba, abba, abbba по шаблону: буква 'a', буква 'b' любое количество раз (в том числе ниодного раза), буква 'a'.
task6.Дана строка 'aa aba abba abbba abca abea'. Регулярка, которая найдет строки aa, aba по шаблону: буква 'a', буква 'b' один раз или ниодного, буква 'a'
task7.Дана строка 'aa aba abba abbba abca abea'. Регулярка, которая найдет строки aa, aba, abba, abbba, не захватив abca abea.
task8Дана строка 'ab abab abab abababab abea'. Регулярка, которая найдет строки по шаблону: строка 'ab' повторяется 1 или более раз.
